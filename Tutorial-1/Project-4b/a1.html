This is a small piece of text, to show the behaviour of the css display property. There are mainly four important values for display.
  • display: inline
  • display: block
  • display: inline-block
  • display: none
In HTML elements have mostly a predefined display value. For example: the DIV container is a block element, as well the UL element or the text block element P or the heading H1. That means they are defining a new start and end of a line, when displayed in a browser
Inline elements are SPAN, A, B or IMG. They do not start a new line. With css you can change this behaviour.
